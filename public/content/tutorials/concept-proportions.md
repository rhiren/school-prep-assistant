# Proportions

## 1. What is this concept?

A proportion tells you that two ratios are equal. It helps you compare two situations that follow the same pattern.

You can see proportions in real life with recipes, maps, and scale drawings. For example, if 2 cups of rice need 3 cups of water, then 4 cups of rice need 6 cups of water to keep the same mix.

## 2. What does it mean?

It means two comparisons match.

For example, `2/3 = 4/6` means:
"For every 2 red apples, there are 3 green apples" and
"For every 4 red apples, there are 6 green apples."

Both situations show the same relationship, just with different amounts.

## 3. Key Ideas

- A proportion is true only when the two ratios are equal.
- Order matters. `2/3` is not the same as `3/2`.
- You must compare the same kinds of things in the same order on both sides.
- Equivalent ratios make true proportions because they show the same comparison.

## 4. How to Solve (Step-by-Step)

1. Identify what is being compared.
2. Determine the type of relationship.
   Is it part-to-part, part-to-whole, or a scale problem?
3. Set up the ratios in the same order.
4. Solve by finding the matching scale factor or by using multiplication and division carefully.
5. Check your answer.
   Ask: do both ratios still mean the same thing?

## 5. Why This Works

A proportion is like a recipe.

If you double every ingredient, the taste stays the same.
If you change only one ingredient, the recipe changes.

That is why both parts of a ratio must change together in the same way. When they do, the relationship stays equal.

## 6. Common Mistakes

- Reversing the order.
  If the first ratio is apples to oranges, the second ratio must also be apples to oranges.

- Mixing part-to-part and part-to-whole.
  Do not compare blue marbles to red marbles in one ratio and blue marbles to all marbles in the other.

- Changing only one number correctly.
  If one part doubles, the other part must also double.

- Using the wrong operation.
  Many proportion problems need multiplication or division, not addition.

- Forgetting to check if the answer makes sense.
  If the numbers get smaller when the ratio should get larger, something is probably wrong.

## 7. Examples (CRITICAL)

### Example 1 (easy)

Does `3/5 = 6/10` make a true proportion?

Step 1: Compare the ratios.
The first ratio is `3/5`.
The second ratio is `6/10`.

Step 2: Check if one is a scaled version of the other.
`3 x 2 = 6`
`5 x 2 = 10`

Step 3: Decide.
Both numbers were multiplied by 2.

Answer:
Yes, it is a true proportion.

### Example 2 (medium)

If `4/7 = x/21`, what is `x`?

Step 1: Look at the denominator.
`7` becomes `21`.

Step 2: Find the scale factor.
`21 ÷ 7 = 3`

Step 3: Apply the same change to the numerator.
`4 x 3 = 12`

Step 4: Check.
`12/21` simplifies to `4/7`

Answer:
`x = 12`

### Example 3 (tricky)

A student says `2/3 = 4/5` because both numbers on the right are bigger.

What is the mistake?

Step 1: Check the scaling.
`2 x 2 = 4`
But `3 x 2 = 6`, not `5`

Step 2: Compare the ratios.
`2/3` and `4/5` are not equal.

Step 3: Explain the trap.
The student noticed both numbers got bigger, but bigger numbers do not always make an equivalent ratio.

Answer:
The mistake is that the two parts were not scaled by the same factor.

## 8. Real-Life Connection

Maps use proportions all the time.

If 1 inch on a map stands for 5 miles in real life, then 2 inches stands for 10 miles. The map and the real distance stay in proportion because both measurements grow by the same factor.
