# Solving Proportions

## 1. What is this concept?

Solving proportions means finding a missing number in two equal ratios. You use it when two situations follow the same pattern.

For example, if 2 notebooks cost 6 dollars, then 5 notebooks should cost a matching amount. Solving the proportion helps you find that missing value.

## 2. What does it mean?

It means two comparisons are equal, but one number is missing.

For example:
`2/3 = x/12`

This means:
"2 out of 3" matches "x out of 12."
Your job is to find the number that keeps the relationship the same.

## 3. Key Ideas

- Keep the ratios in the same order on both sides.
- A true proportion shows equal ratios.
- You can solve by scaling or by using multiplication and division carefully.
- Check your answer by putting it back into the proportion.

## 4. How to Solve (Step-by-Step)

1. Identify what is being compared.
2. Decide what kind of relationship it is.
   Is it part-to-part, part-to-whole, or a scale problem?
3. Set up the proportion in the same order.
4. Solve for the missing value.
5. Check that both ratios are equal.

## 5. Why This Works

A proportion is like making a bigger or smaller copy.

If the copy keeps the same shape, every part must grow or shrink in the same way. Solving the proportion finds the missing piece that keeps the pattern balanced.

## 6. Common Mistakes

- Reversing one ratio.
  The order must stay the same on both sides.

- Changing only one number by the correct factor.
  Both parts must match the same scale.

- Multiplying or dividing the wrong numbers.
  Read carefully before you solve.

- Forgetting to check the answer.
  A quick check can catch a setup mistake.

## 7. Examples (CRITICAL)

### Example 1 (easy)

Solve `3/4 = x/8`

Step 1: Look at the denominator.
`4` became `8`

Step 2: Find the factor.
`8 ÷ 4 = 2`

Step 3: Scale the numerator.
`3 x 2 = 6`

Final answer:
`x = 6`

### Example 2 (medium)

If 5 notebooks cost 15 dollars, how much do 8 notebooks cost?

Step 1: Set up the proportion.
`5/15 = 8/x`

Step 2: Look for the scale.
`8 ÷ 5 = 1.6`

Step 3: Scale the cost.
`15 x 1.6 = 24`

Step 4: Check.
`5/15 = 1/3`
`8/24 = 1/3`

Final answer:
`24 dollars`

### Example 3 (tricky)

A student says `2/5 = 4/9` because the numerator doubled.

What is the mistake?

Step 1: Check the numerator.
`2 x 2 = 4`

Step 2: Check the denominator.
`5 x 2 = 10`, not `9`

Step 3: Explain the trap.
Only one part scaled correctly.

Final answer:
The denominator did not change by the same factor.

## 8. Real-Life Connection

Maps use solving proportions.

If 1 inch on a map stands for 4 miles, then 3 inches stands for 12 miles. If the map length changes, a proportion helps you find the real distance.
