# Scale Drawings

## 1. What is this concept?

A scale drawing is a picture that shows a real object using a smaller or larger size, but it keeps the same shape and the same proportional relationship.

Architects, map makers, and engineers use scale drawings so they can work with sizes that are easier to measure.

## 2. What does it mean?

It means every length in the drawing matches a real length using the same scale.

For example, if `1 inch = 4 feet`, then:

- `2 inches = 8 feet`
- `3 inches = 12 feet`

Every drawing length and real length must stay in proportion.

## 3. Key Ideas

- A scale compares drawing length to real length.
- The same scale factor must be used for every matching side.
- You can solve scale drawing problems with multiplication, division, or proportions.
- Units matter. Compare inches to inches or feet to feet in a consistent way.
- If one side does not follow the scale, the picture is not a correct scale drawing.

## 4. How to Solve (Step-by-Step)

1. Identify the scale.
   Ask: what drawing length matches what real length?

2. Decide what you need to find.
   Are you solving for the drawing size or the real size?

3. Set up a proportion in the same order.
   For example:
   `drawing / real = drawing / real`

4. Solve using the same scale factor on both sides.

5. Check whether the answer makes sense.
   If the real object is larger, the real measurement should usually be larger than the drawing measurement.

## 5. Why This Works

Scale drawings work because similar figures keep the same ratio between corresponding lengths.

If one side doubles, the matching side must also double.

That is why a consistent scale can represent a much larger or much smaller object without changing its shape.

## 6. Common Mistakes

- Mixing the order of the ratio.
  If the scale is drawing to real, keep that order every time.

- Using different units without thinking.
  Convert units first if needed.

- Changing one side by the wrong scale factor.
  Every matching length must use the same factor.

- Adding instead of multiplying.
  Scale drawing problems usually depend on multiplication or division, not addition.

- Forgetting to ask whether the answer should be bigger or smaller.

## 7. Examples (CRITICAL)

### Example 1 (easy)

The scale is `1 inch = 5 feet`.
If a wall is `3 inches` long in the drawing, how long is it in real life?

Step 1: Compare the drawing length to the scale.
`3 inches` is `3` times `1 inch`.

Step 2: Multiply the real length by the same factor.
`5 feet x 3 = 15 feet`

Answer:
The real wall is `15 feet` long.

### Example 2 (medium)

The scale is `2 centimeters = 6 meters`.
If the real playground is `18 meters` long, how long should the drawing be?

Step 1: Compare the real lengths.
`18 meters` is `3` times `6 meters`.

Step 2: Multiply the drawing length by the same factor.
`2 centimeters x 3 = 6 centimeters`

Answer:
The drawing should be `6 centimeters` long.

### Example 3 (tricky)

A student says a scale of `1 inch = 4 feet` means `5 inches = 9 feet` because both numbers got bigger.

What is the mistake?

Step 1: Check the scale factor.
Going from `1 inch` to `5 inches` means multiplying by `5`.

Step 2: Apply the same factor to the real length.
`4 feet x 5 = 20 feet`

Step 3: Explain the mistake.
The student did not use the same scale factor on both sides.

Answer:
The mistake is that `9 feet` does not keep the same proportional relationship.

## 8. Real-Life Connection

Blueprints for buildings are scale drawings.

A designer may draw a room that is only a few inches wide on paper, but the scale tells builders the real room might be many feet wide.

## 9. Quick Practice

1. Easy: If `1 inch = 3 feet`, how many feet does `4 inches` represent?
2. Medium: If `2 cm = 8 m`, how many centimeters should represent `20 m`?
3. Slightly tricky: Why is `3 inches = 10 feet` not correct for a scale of `1 inch = 4 feet`?

## 10. Answers to Quick Practice

1. `12 feet`
2. `5 cm`
3. It does not use the same scale factor on both sides
