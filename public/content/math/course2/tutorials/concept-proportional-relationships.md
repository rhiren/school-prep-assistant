# Proportional Relationships

## 1. What is this concept?

A proportional relationship happens when two quantities always change together by the same constant factor.

If one value doubles, the other value also doubles. If one value is multiplied by 3, the other value is multiplied by 3 too.

## 2. What does it mean?

It means the ratio between the two quantities stays the same.

For example, if `y = 4x`, then:

- when `x = 1`, `y = 4`
- when `x = 3`, `y = 12`
- when `x = 5`, `y = 20`

The relationship is proportional because `y/x` is always `4`.

## 3. Key Ideas

- A proportional relationship has a constant ratio.
- You can check proportionality with a table, graph, equation, or word problem.
- In a proportional relationship, the graph is a straight line through the origin.
- The equation of a proportional relationship has the form `y = kx`.
- The number `k` is the constant of proportionality.

## 4. How to Solve (Step-by-Step)

1. Compare matching values.
   Ask: does the ratio stay the same each time?

2. Look for the constant of proportionality.
   Divide `y` by `x` when `x` is not `0`.

3. Check the representation.
   - Table: do all rows keep the same ratio?
   - Graph: does the line go through the origin?
   - Equation: can it be written as `y = kx`?

4. Decide whether the relationship is proportional.

5. If it is proportional, identify `k`.

## 5. Why This Works

A proportional relationship is built on equal ratios.

If the ratio between two quantities stays constant, then every pair of values describes the same multiplicative comparison.
That is why tables, graphs, and equations can all represent the same proportional relationship.

## 6. Common Mistakes

- Looking at differences instead of ratios.
- Forgetting that the graph must pass through the origin.
- Mixing up `k` with the input value.
- Assuming every straight-line graph is proportional.
- Ignoring a row that breaks the constant ratio.

## 7. Examples (CRITICAL)

### Example 1 (easy)

Table:

- `x = 1, y = 3`
- `x = 2, y = 6`
- `x = 5, y = 15`

Check the ratio `y/x`:

- `3/1 = 3`
- `6/2 = 3`
- `15/5 = 3`

Answer:
This is proportional, and the constant of proportionality is `3`.

### Example 2 (medium)

Equation: `y = 2.5x`

Because the equation is already in the form `y = kx`, the relationship is proportional.

Answer:
Yes, it is proportional, and `k = 2.5`.

### Example 3 (tricky)

A student says the table below is proportional because the values increase by the same amount:

- `x = 1, y = 5`
- `x = 2, y = 9`
- `x = 3, y = 13`

The differences in `y` are always `4`, but the ratios are:

- `5/1 = 5`
- `9/2 = 4.5`
- `13/3 is about 4.33`

Answer:
The relationship is **not** proportional because the ratio is not constant.

## 8. Real-Life Connection

If each notebook costs the same amount, then total cost and number of notebooks form a proportional relationship.

If one notebook costs `$2`, then:

- 3 notebooks cost `$6`
- 5 notebooks cost `$10`

The constant of proportionality is the price per notebook.
