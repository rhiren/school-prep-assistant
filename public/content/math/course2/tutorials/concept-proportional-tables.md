# Proportional Tables

## 1. What is this concept?

A proportional table is a table of values that shows a proportional relationship.

That means every pair of values in the table keeps the same ratio.

## 2. What does it mean?

It means one quantity changes by the same multiplicative rule every time.

For example, if each value in the second column is 4 times the value in the first column, then the table is proportional.

## 3. Key Ideas

- In a proportional table, the ratio between corresponding values stays constant.
- You can test proportionality by dividing the output by the input.
- Missing values can be found by scaling up, scaling down, or using a unit rate.
- A table is not proportional if even one row breaks the constant ratio.
- Proportional tables connect directly to equations like y = kx.

## 4. How to Solve (Step-by-Step)

1. Compare each pair of matching values.
2. Divide output by input to look for a constant ratio.
3. If the ratio matches in every row, the table is proportional.
4. Use that ratio to fill in any missing values.
5. Check that your answer keeps the same relationship across the whole table.

## 5. Why This Works

A proportional relationship is built from equal ratios.

A table makes those equal ratios easy to compare row by row. If all the rows match the same multiplicative rule, the table represents one consistent proportional relationship.

## 6. Common Mistakes

- Comparing differences instead of ratios.
- Using one correct row and ignoring a row that breaks the pattern.
- Multiplying one column without multiplying the other by the same factor.
- Filling a missing value by adding instead of scaling.
- Forgetting that 0 pairs naturally fit a proportional table when the relationship includes the origin.

## 7. Examples (CRITICAL)

### Example 1 (easy)

Table:

- x = 2, y = 10
- x = 4, y = 20
- x = 7, y = 35

Check the ratio y/x:

- 10/2 = 5
- 20/4 = 5
- 35/7 = 5

Answer:
The table is proportional, and the constant of proportionality is 5.

### Example 2 (medium)

Table:

- x = 3, y = 12
- x = 5, y = 20
- x = 8, y = ?

The ratio is 12/3 = 4 and 20/5 = 4.
So y must be 8 x 4 = 32.

Answer:
The missing value is 32.

### Example 3 (tricky)

Table:

- x = 2, y = 8
- x = 4, y = 16
- x = 6, y = 25

The first two rows have ratio 4, but 25/6 is not 4.

Answer:
The table is not proportional because one row breaks the constant ratio.

## 8. Real-Life Connection

If each notebook costs the same amount, a table of notebooks and total cost is proportional.

You can use the table to predict new costs or check whether the price stayed consistent.
