# Proportional Equations

## 1. What is this concept?

A proportional equation is an equation that shows a proportional relationship.

It has the form:

`y = kx`

where `k` is the constant of proportionality.

## 2. What does it mean?

It means one quantity is always a constant multiple of the other.

If `k = 4`, then every output is `4` times the input.

For example:

- if `x = 2`, then `y = 8`
- if `x = 5`, then `y = 20`

## 3. Key Ideas

- A proportional equation has the form `y = kx`.
- The value `k` is the unit rate or constant of proportionality.
- There is no added number like `+ 3` in a proportional equation.
- You can find `k` by dividing `y` by `x`.
- Tables, graphs, and word situations can all be written as proportional equations.

## 4. How to Solve (Step-by-Step)

1. Check whether the relationship is proportional.
2. Find the constant of proportionality.
   Divide `y` by `x`.
3. Write the equation in the form `y = kx`.
4. Use the equation to find missing values.
5. Check your result by substituting a known pair of values.

## 5. Why This Works

In a proportional relationship, the ratio `y/x` stays constant.

That constant ratio is called `k`, so every output can be written as `k` times the input.
That is why the equation is always `y = kx`.

## 6. Common Mistakes

- Writing `y = x + k` instead of `y = kx`.
- Forgetting to divide `y` by `x` to find the constant.
- Calling an equation proportional when it has a nonzero added value.
- Mixing up the input and output variables.
- Using a point that does not fit the relationship.

## 7. Examples (CRITICAL)

### Example 1 (easy)

Table:

- `(1, 5)`
- `(3, 15)`
- `(6, 30)`

Find `k`:

- `5/1 = 5`
- `15/3 = 5`

Answer:
The equation is `y = 5x`.

### Example 2 (medium)

A graph goes through `(0, 0)` and `(4, 10)`.

Find `k`:

- `10/4 = 2.5`

Answer:
The equation is `y = 2.5x`.

### Example 3 (tricky)

Equation: `y = 4x + 2`

Answer:
This is **not** a proportional equation because it has an added `+ 2`.

## 8. Real-Life Connection

If a gym charges the same amount per class, the total cost can be written with a proportional equation.

If each class costs `$7`, then total cost `y` for `x` classes is:

`y = 7x`

## 9. Quick Practice

1. Easy: What is the equation if the constant of proportionality is `3`?
2. Medium: If a proportional relationship goes through `(5, 12.5)`, what is the equation?
3. Slightly tricky: Why is `y = 6x + 1` not proportional?

## 10. Answers to Quick Practice

1. `y = 3x`
2. `y = 2.5x`
3. Because proportional equations cannot have an added value.
