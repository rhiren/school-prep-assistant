# Proportional Graphs

## 1. What is this concept?

A proportional graph shows a proportional relationship on a coordinate plane.

It is a straight line that passes through the origin, which is the point `(0, 0)`.

## 2. What does it mean?

It means the two quantities change together by the same constant factor.

If the graph includes points like `(1, 4)`, `(2, 8)`, and `(5, 20)`, then the ratio `y/x` stays the same.

## 3. Key Ideas

- A proportional graph is a straight line through the origin.
- The constant of proportionality is the same as the unit rate.
- If a point `(x, y)` is on the graph, then `y/x = k`.
- A steeper proportional graph has a larger constant of proportionality.
- A line that does not pass through the origin is not proportional.

## 4. How to Solve (Step-by-Step)

1. Check whether the graph is a straight line.
2. Check whether it passes through `(0, 0)`.
3. Choose a clear point on the graph, like `(2, 10)`.
4. Divide `y` by `x` to find the constant of proportionality.
5. Use that constant to describe the relationship or find missing values.

## 5. Why This Works

In a proportional relationship, every point must keep the same multiplicative ratio.

That creates a straight line through the origin because when `x = 0`, `y` must also be `0`, and every other point follows the rule `y = kx`.

## 6. Common Mistakes

- Thinking every straight line is proportional.
- Forgetting to check whether the line goes through the origin.
- Using the difference between coordinates instead of the ratio `y/x`.
- Mixing up a steeper graph with a graph that only has larger numbers.
- Choosing a point that is not actually on the graph.

## 7. Examples (CRITICAL)

### Example 1 (easy)

A graph goes through `(0, 0)`, `(1, 3)`, and `(4, 12)`.

Check the ratio:

- `3/1 = 3`
- `12/4 = 3`

Answer:
The graph is proportional, and the constant of proportionality is `3`.

### Example 2 (medium)

A graph goes through `(0, 0)` and `(5, 20)`.

Find the constant:

- `20/5 = 4`

Answer:
The equation is `y = 4x`.

### Example 3 (tricky)

A line goes through `(0, 2)`, `(2, 6)`, and `(4, 10)`.

The graph is straight, but it does not pass through `(0, 0)`.

Answer:
The relationship is **not** proportional.

## 8. Real-Life Connection

If a bike rental costs the same amount each hour, a graph of hours and total cost is proportional.

The slope of the graph shows the cost per hour, and the graph should begin at `(0, 0)` if there is no starting fee.

## 9. Quick Practice

1. Easy: Why must a proportional graph pass through `(0, 0)`?
2. Medium: If a graph goes through `(2, 8)`, what is the constant of proportionality?
3. Slightly tricky: Why is a line through `(0, 5)` not proportional?

## 10. Answers to Quick Practice

1. Because when `x = 0`, a proportional relationship has `y = 0`.
2. `4`
3. Because it does not pass through the origin.
